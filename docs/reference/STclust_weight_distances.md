# STclust_weight_distances: Create weighted distance matrices

<div class="ref-description section level2">

Combine scaled expression and spatial distance matrices using
user-specified weights

</div>

<div class="section level2">

## Usage

<div class="sourceCode">

``` r
STclust_weight_distances(scaled_dists, ws)
```

</div>

</div>

<div class="section level2">

## Arguments

-   scaled_dists:

    list of scaled distance matrices (expression + spatial)

-   ws:

    vector of spatial weights to apply

</div>

<div class="section level2">

## Value

list of weighted distance matrices

</div>
