# raise_err: Raise error from exception database

<div class="ref-description section level2">

Stops with an error message from the spatialGE exception database

</div>

<div class="section level2">

## Usage

<div class="sourceCode">

``` r
raise_err(err_code = NULL, samplename = NULL)
```

</div>

</div>

<div class="section level2">

## Arguments

-   err_code:

    error code to look up

-   samplename:

    optional sample name to include in error message

</div>
