# STgradient_prepare_distances: Prepare distance data

<div class="ref-description section level2">

Calculate and categorize distances for gradient analysis

</div>

<div class="section level2">

## Usage

<div class="sourceCode">

``` r
STgradient_prepare_distances(x, sample_name, annot, ref, exclude)
```

</div>

</div>

<div class="section level2">

## Arguments

-   x:

    STlist object

-   sample_name:

    name of sample to process

-   annot:

    annotation column name

-   ref:

    reference cluster

-   exclude:

    optional cluster to exclude

</div>

<div class="section level2">

## Value

list with distance data frames
