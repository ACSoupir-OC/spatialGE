# expandSparse: Convert sparse matrix to dense data frame

<div class="ref-description section level2">

Expands a sparse matrix to a dense data frame

</div>

<div class="section level2">

## Usage

<div class="sourceCode">

``` r
expandSparse(sparsedMatrix)
```

</div>

</div>

<div class="section level2">

## Arguments

-   sparsedMatrix:

    a sparse matrix to convert

</div>

<div class="section level2">

## Value

a dense data frame
