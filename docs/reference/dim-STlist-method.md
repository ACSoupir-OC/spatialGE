# dim: Prints the dimensions of count arrays within an STList object.

Returns the number of genes and spots for each array within an STList
object

## Usage

``` r
# S4 method for class 'STlist'
dim(x)
```

## Arguments

- x:

  an STList object to show summary from.

## Details

This function takes an STList and prints the number of genes (rows) and
spots (columns) of each spatial array within that object.
