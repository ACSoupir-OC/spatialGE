# spatial_metadata: Prints the names of the available spot/cell annotations

<div class="ref-description section level2">

returns the names of the annotations in the `x@spatial_meta` slot.

</div>

<div class="section level2">

## Usage

<div class="sourceCode">

``` r
spatial_metadata(x)
```

</div>

</div>

<div class="section level2">

## Arguments

-   x:

    an STList object

</div>

<div class="section level2">

## Value

a list of character vectors containing the column names of
`x@spatial_meta`

</div>

</div>
