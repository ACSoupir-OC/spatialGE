# STgradient_identify_variable_genes: Identify variable genes

Select top variable genes for analysis

## Usage

``` r
STgradient_identify_variable_genes(x, sample_name, dists_summ, topgenes)
```

## Arguments

- x:

  STlist object

- sample_name:

  sample name

- dists_summ:

  distance summary data frame

- topgenes:

  number of top genes to select

## Value

vector of gene names
