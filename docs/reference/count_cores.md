# count_cores: Determine optimal number of cores for parallelization

<div class="ref-description section level2">

Determines the number of cores to use for parallelization

</div>

<div class="section level2">

## Usage

<div class="sourceCode">

``` r
count_cores(n)
```

</div>

</div>

<div class="section level2">

## Arguments

-   n:

    an integer representing the number of units to process

</div>

<div class="section level2">

## Value

the number of cores to use (max = half of available cores)

</div>
